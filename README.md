# nextjs-sandbox

A simple setup to play around with Next.js

This is a starter template for [Learn Next.js](https://nextjs.org/learn).


## Development

```sh
yarn

yarn dev
```

## Experiencing Static Site Generation vs. Server-Side Generation

```sh
yarn build

yarn start
```
